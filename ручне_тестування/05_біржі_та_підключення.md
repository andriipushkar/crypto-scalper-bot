## 3. API підключення

### TC-104: Налаштування API credentials
**Кроки:**
1. Відкрити Settings -> Exchange API
2. Ввести API Key та Secret

**Очікуваний результат:**
- [ ] Credentials збережено (encrypted)
- [ ] Тест підключення успішний
- [ ] Secret замаскований в UI

### TC-105: Тест з'єднання
**Кроки:**
1. Натиснути "Test Connection"

**Очікуваний результат:**
- [ ] "Connection successful" при валідних ключах
- [ ] "Invalid API key" при невалідних
- [ ] Час відповіді відображається

### TC-106: API permissions check
**Кроки:**
1. При підключенні перевіряються permissions

**Очікуваний результат:**
- [ ] Перевірено: read, trade, withdraw (якщо потрібно)
- [ ] Попередження якщо недостатньо прав

### TC-107: Multiple exchanges підключення
**Кроки:**
1. Налаштувати API для декількох бірж

**Очікуваний результат:**
- [ ] Всі біржі підключені
- [ ] Можна перемикатись між ними
- [ ] Балуновання показує суму
